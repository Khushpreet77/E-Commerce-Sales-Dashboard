# E-Commerce-Sales-Dashboard
E-Commerce Sales Dashboard using power bi
Project Overview

This project is an interactive Mobile Sales Dashboard created using Power BI to analyze mobile sales performance, customer trends, revenue insights, and business growth. The dashboard helps in understanding sales patterns and making data-driven decisions through visual reports and KPIs.

Features
  Interactive and user-friendly dashboard
  Sales performance analysis
  Revenue and profit insights
  Customer and product trend analysis
  Dynamic filters and slicers
  KPI tracking and business reporting
  Data visualization using charts and graphs
Tools & Technologies Used
  Power BI
  DAX
  Power Query
  Data Modeling
  Excel
Dashboard Insights
  Total sales and revenue analysis
  Product-wise and category-wise performance
  Customer purchasing trends
  Monthly and yearly sales comparison
  Business growth tracking using KPIs
Learning Outcomes
  Data cleaning and transformation
  Dashboard designing in Power BI
  DAX calculations and measures
  Data modeling and relationships
  Business data analysis and visualization
